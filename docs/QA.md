# QA - 手動確認チェックリスト

## 初回セットアップ
- [ ] 生活時間未設定時にセットアップが促される
- [ ] 平日睡眠を設定できる（開始/終了）
- [ ] 休日睡眠のON/OFFができる
- [ ] 固定予定を追加/削除できる

## 週計画
- [ ] 週切替で表示期間が変わる
- [ ] 週未作成の空状態が表示される
- [ ] 週計画を作成できる
- [ ] 学習ブロックを30分単位で追加できる
- [ ] 終了時刻が翌日扱いになる場合の表示がある
- [ ] 完了ステータスを変更できる
- [ ] 今週の学習可能時間が表示される
- [ ] 今週の実績合計が表示される
- [ ] カテゴリ別内訳が表示される

## 自動生成
- [ ] 教材がある場合に自動生成できる
- [ ] 再生成で追加/置換が選べる
- [ ] 手動ブロックは維持される

## 教材
- [ ] 教材追加時にカテゴリが必須
- [ ] 教材追加時に新規カテゴリを作成できる
- [ ] 教材の編集/削除ができる

## 学習履歴
- [ ] 期間選択ができる
- [ ] 期間内のサマリが表示される
- [ ] 週一覧から週計画へ遷移できる

## 表示/共通
- [ ] 30分刻みが「6:30」形式で表示される
- [ ] 1時間境界が実線、30分境界が破線
- [ ] 空/読み込み/エラーで真っ白にならない
