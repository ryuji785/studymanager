# FLOWS - 学習計画のユーザーフロー

## 1. 初回セットアップ
1. 週計画画面を開く
2. 生活時間が未設定の場合、セットアップダイアログが表示
3. 平日睡眠を必須入力
4. 休日用を必要に応じてON
5. 固定予定（通勤・食事など）を追加
6. 保存 → 週計画へ戻る

## 2. 週計画作成
1. 週切替（期間セレクタ）で対象週を選択
2. 週計画が未作成なら「週計画を作成」を押す
3. 「＋学習ブロック追加」から手動入力
   - カテゴリ＋時間は必須
   - 教材は任意

## 3. 自動生成/再生成（任意）
1. 教材が登録済みであることを確認
2. 「今週タスクを自動生成」→ 追加
3. 「再生成」→ 追加 or 置換 を選択
4. 手動ブロックは維持される

## 4. 完了チェックと実績確認
1. タイムテーブルのブロックを編集
2. 状態を「完了」に変更
3. 週計画サマリーで実績合計とカテゴリ内訳を確認

## 5. 学習履歴
1. メニューから学習履歴へ
2. 期間を選択
3. 週一覧から週計画へ戻る

## 6. 教材とカテゴリ管理
1. 教材を追加（カテゴリ必須）
2. カテゴリがなければその場で追加
3. 教材の編集/削除
